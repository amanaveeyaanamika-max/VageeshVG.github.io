# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Features smooth animations, mobile-friendly design, and interactive elements.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Engaging scroll animations and transitions
- **Interactive Navigation**: Mobile-friendly hamburger menu
- **Project Showcase**: Display your best work with project cards
- **Skills Section**: Visual representation of your technical skills
- **Contact Form**: Functional contact form for visitor inquiries
- **Modern UI**: Clean, professional design with gradient accents

## 📝 Customization Guide

### 1. Update Personal Information

Edit `index.html` and replace:
- Your name in the hero section
- Email, phone, and location in the contact section
- Social media links (GitHub, LinkedIn, Twitter)

### 2. Add Your Projects

In the projects section, update:
- Project titles and descriptions
- Technology tags
- Project links

### 3. Adjust Skills

Modify skill bars in the skills section:
- Change skill names
- Adjust progress bar widths (style="width: XX%")

### 4. Color Scheme

Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* Customize other colors */
}
```

## 🌐 Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push your code to the repository
3. Go to Settings > Pages
4. Select main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`
